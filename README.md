# p5.js Photo Collage

Generative art collage using random photos from your Photos Library.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create your config file:
   ```bash
   cp config.example.js config.js
   ```
   Edit `config.js` and set your photos directory path.

3. Start the server:
   ```bash
   npm start
   ```

4. Open your browser to:
   ```
   http://localhost:3000
   ```

The app will display 3 random photos from your Photos Library with generative bezier curve masks.
